# Market Tape

Live market dashboard for cloud infrastructure and FinOps teams.

Deployed at [market-tape.cloudandcapital.com](https://market-tape.cloudandcapital.com).

## What it does

Market Tape provides a daily read on what's happening in markets and what it means for cloud and AI infrastructure spending. Updated every 30 minutes during market hours.

## Sections

| Section | Description |
|---|---|
| **Claude's Morning Brief** | AI-generated 3-paragraph market analysis via Claude API |
| **Market Status** | Exposure guidance (Risk-On / Hold / Defensive), trend, breadth, momentum score |
| **Sector Leaders** | Top performers by 1-month relative strength across sectors and countries |
| **AI & Infrastructure** | SMH, SOXX, IGV, SKYY, AIQ, PAVE — prices, % change, sparklines |
| **Momentum Leaderboard** | Top 10 leaders + top 5 laggards with trend grades |

## Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS v4
- **AI:** Anthropic Claude API (morning brief generation)
- **Data:** Static JSON from [cloudandcapital/market-tape](https://github.com/cloudandcapital/market-tape), refreshed every 30 min via automated pipeline
- **Deployment:** Vercel

## Development

```bash
npm install
npm run dev      # localhost:3000
npm run build
```

## Environment variables

```
ANTHROPIC_API_KEY=sk-ant-...
```

Set in Vercel project settings for production. Required for the Morning Brief feature.

## Data pipeline

Market data is generated by the Python pipeline in [cloudandcapital/market-tape](https://github.com/cloudandcapital/market-tape) and written to `data/meta.json` and `data/snapshot.json`. This app fetches those files directly from GitHub raw URLs with 30-minute ISR revalidation.
